Nitro 是 Nuxt 3 的服务器引擎，支持多种运行环境。它允许你创建高性能的服务器端应用程序，处理各种 HTTP 请求，动态生成内容等。

2. 基本概念

  • 处理程序：函数，用于处理特定的 HTTP 请求或路由。
  • 路由：URL 路径，用于匹配处理程序。
  • 中间件：在处理请求前处理某些操作的函数，通常用于身份验证或请求修改。
