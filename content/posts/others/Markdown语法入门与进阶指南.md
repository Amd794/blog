## 一、Markdown简介

Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯（john Gruber）。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML（或者HTML）文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。

## 二、Markdown的基本语法

1.  标题：

使用#号创建标题，其数量代表了标题的级别。

```
# 一级标题
## 二级标题
### 三级标题
```

1.  文本样式：

```
**加粗**  
*斜体*  
_下划线_
```

1.  列表：

```
无序列表：
* 项目1
* 项目2

有序列表：
1. 项目1
2. 项目2
```

1.  引用：

```
单引号：'文字'
双引号："文字"
```

1.  链接：

```
[链接文字](链接地址)
```

1.  图片：

```
![图片描述](图片地址)
```
